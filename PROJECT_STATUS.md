# LinkedIn Job Assistant - 项目进度

## 项目概览
LinkedIn Job Assistant是一个AI驱动的求职助手工具，集成了简历优化、职位追踪和申请材料管理功能。

## 技术架构
- **前端**: Next.js 14 + TypeScript + Tailwind CSS
- **后端**: Next.js API Routes
- **数据库**: Supabase
- **AI集成**: Claude API (Anthropic)
- **国际化**: 自定义i18n系统
- **组件库**: shadcn/ui
